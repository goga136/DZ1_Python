1.	Запуск тестов с генерацией отчета.
Для запуска автотестов и сохранения результатов для Allure необходимо выполнить команду из корня проекта:
pytest --alluredir=allure-results
В результате выполнения команды создаётся каталог allure-results, содержащий результаты выполнения тестов.

2.	Просмотр сформированного отчета
Для просмотра сформированного Allure-отчета необходимо выполнить команду:
allure serve allure-results
После выполнения команды отчет автоматически откроется в браузере.
